# CTD Todo List Project

This project was created as part of CTDs React Course. It's build with React + Vite.

## How to Install

1. Clone the repository
```bash
git clone https://github.com/dmdi-tech/ctd-todo-list.git
cd ctd-todo-list
```

## Install dependencies

```bash
npm install
```

## Start Server
```bash
npm run dev
```

## Click the link in the terminal or paste the following in your browser
```bash
http://localhost:5173/
```
